     This is the readme.text file with short description of the phpBB3 style funcionality and configuration. Modestus is a modern and elegant style from RocketTheme.
     
     1. Installation
     
     Style is installed as the usual phpBB3 style.
     For basic style installation instructions please refer to: http://www.phpbb.com/kb/article/how-to-install-styles-on-phpbb3/
     
     2. Style configuration
     
     phpBB3 style is configured as any other phpBB3 style, however we have made few little additions to the basic prosilver style. . All this configuration options can be found in the "modestus_config.html" file. Near the every option there is short comment explaining how current setting works.
     
     2.1 Other differences
     Logos are located /theme/images/stylex
     
     3. Additional files list:
     
     modestus_config.php - configuration file
     mootools-release-1.11.js - mootools file needed for png transparency fix
     belated-png.js - png transparency fix
     
     4. Sources
     
     Sources are Adobe Fireworks slices, which you can easily modify and adjust to fit your needs.
